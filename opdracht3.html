3. Array van objecten

3.1 Array definieren

Definieer de constante beesten,
met als waarde een array met 3 objecten.
Elk van deze objecten stelt een beest voor en heeft de volgende eigenschappen:

  • voornaam
  • bijnaam
  • leeftijd
  • favorieteVoedsel (Array van 3 gerechten)

Voorbeeld namen:

  • Brinecat, The Twin Mutant
  • Abysstalon, The Venom Feline
  • Brineling, The Primeval Mutant
  • Vapormutant, The Venom Mutant
  • Grossbug, The Quiet Monster

3.2. Array doorlopen

Toon hieronder de namen (voornaam en bijnamen) van de beesten (gebruikmakend van de Array),
zet een <br> na elke naam.
Voorbeeld: Brinecat, The Twin Mutant (21 jaar)

3.3. Methode definieren

Defineer in elk object de methode volledigeNaam(), die een string teruggeeft met hierin de volledige naam van heet beest. Brinecat, The Twin Mutant
Toon hieronder de volledige namen van de beesten (gebruikmakend van de Array en methode),
zet een <br> na elke naam.

3.4. Random beest

Toon bij een druk op de onderstaande knop, de voornaam van een random beest.

3.5. Random voedsel van random beest

Toon bij een druk op de onderstaande knop, een random gerechrt van een random beest.
